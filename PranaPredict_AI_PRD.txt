# 4️⃣ Risk Calculation Logic (No ML Coding)

Example scoring system:

```
BMI > 30 → +20 points
BP > 140/90 → +25 points
Cholesterol High → +20
Smoking → +15
Low Activity → +10
Age > 50 → +10
```

Total score → Convert to 0–100 scale

Risk Levels:

* 0–30 → Low
* 31–60 → Moderate
* 61–100 → High

This is acceptable and explainable in interviews.

# 6️⃣ Deployment Guide

---

# 🔹 Backend Deployment (Render)

1. Push backend to GitHub
2. Go to Render
3. Create New Web Service
4. Connect GitHub repo
5. Set:

* Runtime → Node
* Build Command:

  ```
  npm install
  ```
* Start Command:

  ```
  node server.js
  ```

6. Add Environment Variables:

```
SUPABASE_URL
SUPABASE_ANON_KEY
HUGGINGFACE_API_KEY
PORT=10000
```

7. Deploy

Render will provide:

```
https://your-backend.onrender.com
```

---

# 🔹 Frontend Deployment (Vercel)

1. Push frontend folder to GitHub
2. Go to Vercel
3. Import project
4. Set:

Framework → Vite

Build Command:

```
npm run build
```

Output Directory:

```
dist
```

Add Environment Variable:

```
VITE_BACKEND_URL=https://your-backend.onrender.com
VITE_SUPABASE_URL=xxxx
VITE_SUPABASE_ANON_KEY=xxxx
```

Deploy

## Abstract

**Project Title: PranaPredict AI – An Ayurveda-Inspired AI Health Risk Prediction System**

PranaPredict AI is a full-stack web application designed to assess an individual’s health risk using modern clinical parameters integrated with Ayurveda-inspired wellness interpretation. The system enables users to input key health indicators such as age, BMI, blood pressure, cholesterol levels, smoking habits, and physical activity. Using a structured rule-based risk scoring algorithm, the platform calculates a quantitative health risk score and classifies it into low, moderate, or high risk categories.

The backend is developed using Node.js and Express, while the frontend is built with React (Vite). Supabase is utilized for authentication, database management, and secure storage of user health reports. Instead of developing custom machine learning models, the system integrates open-source large language models through the Hugging Face Inference API to generate personalized health explanations and lifestyle recommendations. These AI-generated insights align with preventive healthcare principles inspired by Ayurveda, focusing on balance, diet regulation, physical activity, and sustainable living practices.

The application is deployed using Render for backend services and Vercel for frontend hosting to ensure scalability and accessibility. The architecture follows secure API design standards, including input validation, environment-based configuration, and role-based data protection.

PranaPredict AI demonstrates practical AI integration, cloud-based deployment, structured database design, and secure full-stack development, offering a preventive healthcare technology solution that emphasizes wellness optimization rather than reactive treatment.
